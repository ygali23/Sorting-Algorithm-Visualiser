**https://ygali23.github.io/Sorting-Algorithm-Visualiser/**

## features
- Compare **1–4 algorithms** side-by-side
- One-click **Solo Algorithm** mode
- Bars mode and Lines mode
- Timeline scrubbing (step-by-step)
- Operation race board (comparisons + swaps)
- Sound feedback on swaps (toggleable)
- Keyboard shortcuts for all major actions
- Value-based rainbow colouring  
  *(smallest = red → largest = violet, dynamic as values move)*

---

## algorithms included
| Algorithm | Time Complexity |
|---------|-----------------|
| Bubble Sort | O(n²) |
| Insertion Sort | O(n²) |
| Selection Sort | O(n²) |
| Quick Sort | Avg: O(n log n), Worst: O(n²) |
| Merge Sort | O(n log n) |
| Heap Sort | O(n log n) |
| Radix Sort (LSD) | O(k·n) |
| TimSort (mini) | O(n log n) |
